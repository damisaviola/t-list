# Todo Management Application

## Tech Stack

### Frontend
- **React** – Library JavaScript untuk membangun antarmuka pengguna berbasis komponen.
- **TypeScript** – Memberikan pengetikan statis pada JavaScript agar kode lebih aman dan terstruktur.
- **Tailwind CSS** – Framework CSS berbasis utility untuk desain antarmuka yang cepat dan responsif.
- **Framer Motion** – Digunakan untuk animasi dan transisi antarkomponen.
- **React Router** – Mengatur navigasi dan routing halaman pada aplikasi frontend.

### Backend
- **Golang (Go)** – Bahasa pemrograman backend untuk membangun REST API dengan performa tinggi.
- **Arsitektur MVC (Model–View–Controller)** – Memisahkan logika bisnis, pengelolaan data, dan alur aplikasi.
- **JWT (JSON Web Token)** – Digunakan untuk autentikasi dan pengamanan endpoint API.
- **MySQL** – Sistem manajemen basis data relasional untuk menyimpan data pengguna dan data todo.
